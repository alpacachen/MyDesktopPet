# 更新日志 / Changelog

所有重要的项目变更都会记录在此文件中。

并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] - 2024-12-30

### 新增
- 🎨 集成 Lottie 动画引擎，支持流畅的矢量动画
- 📁 自定义动画导入功能，支持用户导入 Lottie JSON 文件
- 🔄 等比缩放功能，提供 5 档缩放选项（50%-200%）
- 🎭 可爱的哈士奇应用图标
- 🐾 内置两个可爱小狗动画
- 📋 完整的菜单栏控制系统
- 💾 自定义动画持久化存储到 ~/Library/Application Support

### 改进
- 🏗️ 使用 Swift Package Manager 管理依赖
- 📦 完整的 .app 和 .dmg 打包方案
- 📝 中英文双语 README
- 🛠️ 自动化构建脚本

### 技术栈
- Swift + AppKit (原生 macOS 开发)
- Lottie for iOS (动画渲染)
- Swift Package Manager (依赖管理)
